Installation Components:
------------------------

Web Server:
  - Apache 2.4.58
  - Ports: 80, 443 (Snipe-IT), 8080, 8443 (Kiosk)
  - Modules: SSL, WSGI, Rewrite, Headers

Database:
  - MariaDB 10.11.13
  - Databases: snipeit
  - Users: snipeit_user

Application Server:
  - PHP 8.3 (for Snipe-IT)
  - Python 3.x with WSGI (for Kiosk)
  - Redis Server (caching)

Security:
  - UFW Firewall
  - Self-signed SSL certificates (replaceable with Let's Encrypt)
  - Modern TLS configuration

Applications Included:
----------------------

1. Snipe-IT Asset Management
   - Full application files
   - Complete database with data
   - Public uploads directory
   - Configured for ports 80/443

2. Kiosk Application
   - Instructions to clone from GitHub
   - Apache configuration ready
   - Python virtual environment setup guide
   - Configured for ports 8080/8443

What's NOT Included:
--------------------

- Kiosk source code (clone from your GitHub repository)
- Production SSL certificates (use Let's Encrypt)
- Custom passwords (set during installation)
- Server-specific .env configurations (templates provided)

Total: ~20-25 minutes

System Requirements:
--------------------

- Ubuntu 24.04 LTS (fresh installation recommended)
- 2GB RAM minimum
- 20GB available disk space

Post-Installation Tasks:
------------------------

1. Run mysql_secure_installation
2. Update database credentials in .env files
3. Clone Kiosk app from GitHub
4. Install Kiosk Python dependencies
5. Configure Kiosk .env file
6. Install Let's Encrypt certificates (production)
7. Update IP addresses/domain names in configurations

- All scripts are executable (chmod +x)
- Configuration files retain original permissions
- Database exports are uncompressed SQL
- Application archives exclude cache and log directories
- Self-signed SSL certificates valid for 365 days
- Firewall configured for all required ports
