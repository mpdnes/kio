EMERGENCY PROCEDURES
================================================================================

If something breaks:

1. STAY CALM - Most issues are recoverable

2. Check TROUBLESHOOTING.md for the specific issue

3. Review relevant logs:
   sudo tail -100 /var/log/apache2/error.log

4. Try restarting the affected service:
   sudo systemctl restart apache2

5. If database issue:
   sudo systemctl restart mariadb

6. If still broken:
   - Contact system owner
   - Check documentation
   - Restore from backup if needed

7. Document what happened and how you fixed it
