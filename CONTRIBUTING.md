# Contributing to Equipment Kiosk

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/kiosk.git`
3. Create a virtual environment: `python -m venv venv`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the setup wizard: `python setup.py`

## Development Workflow

1. Create a feature branch: `git checkout -b feature/my-new-feature`
2. Make your changes
3. Run tests (if available)
4. Commit your changes
5. Push to your fork
6. Submit a Pull Request

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable and function names
- Add comments for complex logic

## Reporting Issues

Please use the GitHub Issues tracker to report bugs or request features.
